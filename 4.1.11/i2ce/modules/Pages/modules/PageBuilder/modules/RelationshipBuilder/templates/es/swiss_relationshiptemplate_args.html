Opciones de conversión de plantilla

Formato

Formato de salida. El formato predeterminado es ODT. Si deseas otros formatos, necesitarás tener instalado unoconv.

Opciones de exportación

Opciones de exportación para aplicar a UnoConv. Las opciones disponibles dependen del formato de exportación (p. ej. pdf) que estás usando.

Opciones de conexión

Detalles de la conexión para la ejecución de unoconv en el servidor remoto.

UnoConv Temporary Directory

Specify the temporary directory to process unoconv in. Intended for use with remotely mounted drives.

Campos obligatorios

Si un campo obligatorio no está establecido, la plantilla no se generará.
