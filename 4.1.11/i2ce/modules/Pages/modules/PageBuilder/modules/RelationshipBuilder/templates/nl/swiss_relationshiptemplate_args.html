Template Segment Style

Controls how multiple rows of data are applied to the template. "Global" means that the entire ODT template is repeated. "Report Row" means that there is an expected 
 [!-- BEGIN report_row --] Stuff Goes Here [!-- END report_row --]
section in the ODT template used for each row of data.

Segment Break

Template Conversion Options

Format

Output Format. The default format is ODT. If you would like other formats, you will need to have unoconv installed.

Exportinstellingen

Export options to apply to UnoConv. The available options depend on the export format (e.g. pdf) that you are using.

Connection Options

Connection details for running unoconv on remote server.

UnoConv Temporary Directory

Specify the temporary directory to process unoconv in. Intended for use with remotely mounted drives.

Verplichte velden
