Format d'affichage du formulaire:

Modifier le format dans lequel l’information du formulaire sera affichée dans la page. Utiliser les déterminants du format printf pour ajouter les champs à afficher.

Si vous n'êtes pas familier avec les spécificateurs de format printf, utilisez '%s' pour chacun des champs que vous souhaitez voir affiché. L'ordre de la %s' correspondront à l'ordre des champs que vous avez sélectionné (par exemple le premier %s affichera le premier champ).

Champs affichés:

Les champs pour afficher chaque déterminant du format printf. Chaque déterminant de format dans le Format d’affichage sera remplacé par le champ correspondant. Le nombre de déterminants de format doit correspondre au nombre de champs.
