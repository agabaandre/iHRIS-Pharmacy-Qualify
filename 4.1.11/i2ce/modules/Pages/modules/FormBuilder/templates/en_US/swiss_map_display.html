Meta Data

Field Link

Colon seperate list of forms[+field] to define hierarchy

Display Style

The way that this form is displayed when editing.

Report Display

The report used when populating data for populating drop-downs, tree views. You should consider using this if the field linking is complicated.

Orders

Set the ordering by fields

Printf

Set the printf string use for various forms

PrintF Fields

Set the fields (arguments) used in the printf string for various forms

PrintF Field Styles

Set the display styles used for the various fields

Report View Selection

Specify report view used to select data

Show Limits

Specify whether or not to do not show limits for a form in report view selector
