Metadatos

Mostrar clave

Show the key when displaying the associative list

Permitir nuevas

Allow new keys to be added to the list

Allow Delete

Allows keys to be deleted from the list

Key Sort Order

Ordering to sort keys

Auto Fix Percent

If an entered percentage is less than one, assume that the user entered a ratio rather than a percetage, in which case we should multiply it my 100. For example, if a user enters 0.7, it should be interpreted as 70%.
