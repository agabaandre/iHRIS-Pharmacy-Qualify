Administrar Funciones y Tareas

Aquí podrá administrar las funciones y tareas del sistema.

  • Administrar Funciones

    Un rol se utiliza al crear un usuario. Se le asignara un rol a un usuario. Existe un rol especial 'admin', este es el rol de Administrador. También puede definir que roles tienen los mismos derechos que un específico.

  • Administrar Tareas

    Una tarea es una acción específica realizada por el sistema. Podrás asignar un conjunto de tareas a una función. También podrás crear una nueva tarea agrupando un conjunto de tareas
