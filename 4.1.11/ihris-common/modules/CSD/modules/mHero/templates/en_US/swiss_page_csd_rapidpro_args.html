Page Options

RapidPro Token:

The API authorization token for accessing RapidPro. To get this, click on "Account Details" in RapidPro and then click under "Webhook." The API Token should be indicated there.

Host:

The host server for accessing RapidPro (e.g. https://rapidpro.io)

Organization Short Name:

The shortname (or slug) for the RapidPro Organization. To get this, click on "Account Details" in RapidPro and then click "Your Organization." The short name should be indicated there.

CSD Host:

The URL endpoint used to execute Care Service Discovery (CSD) Find Matching Services requests against.

CSD Host User:

The Username to be used for authentication when executing Care Service Discovery (CSD) Find Matching Services requests against.

CSD Host Password:

The Password to be used for authentication when executing Care Service Discovery (CSD) Find Matching Services requests against.
