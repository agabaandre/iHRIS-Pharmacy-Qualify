Shared Value Set - Meta Data

Display Name Style

The list display style used to populate the @displayName attribute of the concept.

Code Style

The list display style used to populate the @code attribute of the concept.

Code System Style

The list display style used to populate the @codeSystem attribute of the concept. If it is not set, it will default to the value in "Source."

Status

Possible values include 'Active' and 'Inactive'

Source

Source URI

Definition

Type

This describes the type of the value set and should be one of 'Intensional', 'Extensional', or 'Expanded.'

Binding

Optional, but should be one of 'Static' or 'Dynamic' if set.
