Gerenciar Relatórios

Gerenciar os relatórios padronizados disponíveis para usuários e criar novos relatórios personalizáveis.

  • Formular relacionamentos
    Definir como os formulários estão relacionados para uso nos relatórios.
  • Relatórios
    Definir relatórios baseados em relacionamentos previamente definidos. Você pode alterar quais formulários estão disponíveis para cada relatório e cada campo pode ser usado para limitar o relatório.
  • Visualização do relatório
    Define relatórios visualizados com base em relatórios previamente definidos. Você pode alterar quais campos mostrados e também definir uma visualização padronizada para mostrar uma tabela ou limitar resultados quando um usuário acessar a visualização do relatório.
