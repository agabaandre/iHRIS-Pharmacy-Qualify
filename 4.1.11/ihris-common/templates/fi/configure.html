Raporttien hallinta

Hallinnoi käyttäjien käytettävissä olevia vakioraportteja ja luo uusia omia raportteja.

  • Muodosta yhteyksiä
    Määritä, kuinka lomakkeet vaikuttavat käyttöön raporteissa.
  • Raportit
    Määritä raportit, jotka perustuvat aiemmin määriteltyyn suhteeseen. Voit valita, mitkä kentät ovat käytettävissä jokaisessa raportissa ja mitä kenttiä käytetään raportin rajoittamiseen.
  • Raporttinäkymät
    Define report views based on previously defined reports. You can choose which fields to display and also define a default view that display a chart or limited results when a user access the report view.
