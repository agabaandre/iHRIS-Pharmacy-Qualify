보고서 관리

사용자에게 제공하는 표준 보고서를 관리하고 새 사용자 정의 보고서를 만든다.

  • 양식 관계
    어떻게 폼이 리포트에 사용되기 위해 관련이 있는지 정의힌다.
  • 보고서
    Define reports based on previously defined relationships. You can choose which fields are available to each report and what fields can be used to limit the report.
  • 보고서 보기
    Define report views based on previously defined reports. You can choose which fields to display and also define a default view that display a chart or limited results when a user access the report view.
