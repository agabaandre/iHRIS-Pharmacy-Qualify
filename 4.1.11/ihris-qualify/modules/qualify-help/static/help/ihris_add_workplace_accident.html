You can utilize the Current Disciplinary Actions Module to record disciplinary actions.

In the person's record under "Current Disciplinary Actions," click New Action.
The Add/Update Disciplinary Action form opens.

From the drop down menu, select the Action Taken.

Select the Date of Discussion and Start of Applicability, using the date pickers.

Use the text boxes to record People Present and any Notes. Click Confirm and confirm that the information entered is correct. If it is not correct, click Edit to change it. If it is, click Save to save it.
The disciplinary action information that you just entered displays in the employee's record. Additional disciplinary actions can now be added. For each disciplinary action, click New Action. To view a record of all disciplinary actions, click History. If any of the disciplinary action information needs to be changed, click Correct this information beside the incorrect information to edit it.
