You can utilize the Dependents Module to track information on employees' dependents.

In the person's record under "Dependent Information" section, click Add Dependent.
The Add/Update Dependent form opens.

Enter the dependent's Name.

Select the Gender and Date of Birth.

Click Confirm and confirm that the information entered is correct. If it is not correct, click Edit to change it. If it is, click Save to save it.
The dependent information that you just entered displays in the employee's record. Additional dependents can now be added. For each dependent, click Add Dependent and add the new dependent's information. If any of the identification information needs to be changed, click Update This information beside the incorrect information to edit it.
